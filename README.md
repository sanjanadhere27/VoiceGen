# VoiceGen
Python GUI-based Text-to-Speech Converter using Tkinter and pyttsx3 with voice, speed, and volume control.


# 🎤 VoiceGen - Text To Speech Converter

A simple and user-friendly Text to Speech Converter application built using Python.  
This application allows users to convert written text into speech with customizable voice, speed, and volume.

## 🚀 Features
- Convert text into speech instantly
- Male and Female voice options
- Adjustable speech speed
- Volume control
- Clean GUI using Tkinter
- Smooth performance using threading

## 🛠️ Technologies Used
- Python
- Tkinter
- pyttsx3
- Threading

## ▶️ How to Run

1. Install Python
2. Install library:
         pip install pyttsx3
3. Run:
python app.py
